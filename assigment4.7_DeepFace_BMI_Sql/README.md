## Describe

This is a application that utilizes for face analysis and BMR calculator..


## Setup and Installation

* Install Dependencies:

  ```bash
  pip install -r requirements.txt
  ```

* Run

  ```bash
  flask run
  ```
## Docker


  ```bash
  docker build -t ai_web_app .
  ```
